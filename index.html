Layout Elements

Basic Flexboxes

Each of the following can be used as a class for any element, not just div.

  1. First make a parent flexbox element by using one of the following classes on any html element:
    • flexrow-wrap
    • flexrow-nowrap
    • flexcol-wrap
    • flexcol-nowrap
  2. Then you can add one of the following classes to set the alignment along the main-axis of that parent flexbox.
    • just-start
    • just-mid
    • just-end
    • just-left
    • just-right
    • just-stretch
    • just-around
    • just-between
    • just-even
  3. Then you can add one of the following classes to set the alignment along the cross-axis of that parent flexbox.
    • align-start
    • align-mid
    • align-end
    • align-stretch
    • align-anchor
    • align-base
    • align-base-first
    • align-base-last

Empty Split Sections

Each corresponds to a flexbox with div children, and are applied to section tags. The numbers in the boxes shows the ratio of the size of that box to the other boz(es) in the row. The name of these classes all start with "split" followed by the ratio separated with dashes. So the first example would be class="split-1-2". The borders are only enabled here to help visualize, but would not be enabled by default.
